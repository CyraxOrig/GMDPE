# GMDPE — Geometry More Dash Physical Extension

**GMDPE** is a comprehensive hack/utility menu for Geometry Dash 2.2081, developed by **CyraxOrig**.

It includes over 20 features split across four categories:

## 🎮 Gameplay
- **Noclip** — pass through all obstacles; adjustable player opacity
- **Straight Fly** — lock ship to a horizontal path with a smoothness slider (0–100%)
- **Auto Clicker** — configurable CPS auto-click
- **Instant Respawn** — skip the death animation entirely

## ⚡ Performance
- **FPS Bypass** — uncap the frame rate up to 1000 FPS
- **TPS Bypass** — decouple physics ticks from rendering for buttery-smooth play
- **Speedhack** — scale game speed from 0.1× to 10×

## 👁 Visual
- **Show Hitboxes** — see every object's collision box (optional: on death only)
- **Hide Player** — make your icon invisible
- **Solid Wave Trail** — replace the default dotted wave trail with a solid line
- **No Death Effects** — disable explosion particles

## 📊 Level Info
- **CPS Counter** — live clicks-per-second overlay
- **Noclip Death Counter** — track how many times you'd have died
- **Attempt Number** — on-screen attempt display
- **StartPos Switcher** — cycle StartPos objects with arrow keys

## 🔧 Miscellaneous
- **Practice Music Sync** — music plays from the correct timestamp in practice
- **Unlock All Icons (Visual)** — preview all icons locally
- **Ignore ESP / Anti-Cheat** — suppress the cheat indicator dot
